Provocation · 8 Minutes

AI Will Compress Your Margins
Before It Expands Them.

Vote · reveal · argue

👋 Show of hands: who knows inference cost / user / month?

NOT YET
"AI will lower our costs."
Inference is variable now. Many AI features land ~40–60% gross vs ~80% SaaS — often a P&L surprise.
FALSE
"Price per seat, like always."
Seats = access. AI often = work done — seat pricing under-captures value.
FALSE
"Scale fixes the economics."
Scaling a negative unit-econ product scales losses. Fix = architecture + pricing, not volume alone.

Lecture · Bundling Strategy

Leaders, Fillers & Killers

Bundle wrong and you give the farm away. — Madhavan Ramanujam
Leader
What users come for — must-have, high WTP
AI: core intelligence (e.g. GitHub Copilot autocomplete)
Filler
Nice add for marginal price — bumps ARPU
AI: summaries, translations, templates
Killer
Kills the bundle if included — sell as add-on
AI: heavy inference (image gen, agent workflows)

70% rule: If >70% of users use the AI feature → bundle it. <70% with high value → add-on.

Gokul: Raise price on proven value. Don't only shave tokens — do both.

Case Study · 7 Minutes

The Automator That Overcorrected

The Bet
Klarna AI assistant: ~⅔ of support chats in early rollout — press framed as ~700 FTE equivalent. Thesis: automate, cut cost.
The Crack
Token $ scale with usage · edge cases → humans · variable COGS replaced fixed labor → messier forecasting
The Correction
Tiering + copilot for hard cases · narrative shifted to CX (speed, 24/7, languages) not only headcount
