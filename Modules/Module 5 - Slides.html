Lecture · Point 1 of 4

Systems That Compound

The difference between getting bigger and getting better.
System That Scales
More volume. Same quality. Linear cost. Gets bigger, not better.
System That Compounds
Every interaction improves the next. Quality increases. Cost/unit decreases. Gets better AND bigger.
Recursive Learning
System learns from outputs. Corrections refine model.
Cross-Domain Transfer
Learning in one area improves adjacent areas.
Network Intelligence
Patterns from one customer improve all (with privacy).

Gokul: Vertical AI must own the full stack. Service Titan runs 32 products — depth creates compounding horizontal can't match.

3
Governance as Sales Asset

Lecture · Point 3 of 4

Agent Governance

Four dimensions for when agents start making decisions.
Autonomy Boundaries
"Can draft, not send. Can query, not modify." Define the decision space explicitly.
Tool-Calling Safety
Each tool call is a risk surface. Whitelist tools. Rate-limit. Log everything.
Memory Management
What does the agent remember? For how long? Who accesses the memory? HIPAA implications.
Chain Accountability
When agent B fails based on agent A's output — who's responsible? Define at each handoff.

Gokul: Capabilities change every 6 months. Build principle-based frameworks, not model-specific ones.

Lecture · Point 4 of 4

Agentic vs. Traditional ROI

Not all automation is created equal.
Traditional Automation
Predictable cost. Predictable output. Linear ROI. Does exactly what you program. Breaks on exceptions.
Agentic Automation
Variable cost. Probabilistic output. Non-linear ROI. Handles exceptions. Harder to govern.
Task Complexity
Simple → traditional. Complex → agents.
Error Tolerance
Low → governance eats ROI. High → ROI compounds.
Learning Value
Traditional: no. Agents: yes, if captured.
Case Study · 7 Minutes

Internal AI Failure → External Governance Story

The Bet
Samsung engineers used ChatGPT for code review, debugging, meeting summaries. Standard productivity — employees using available tools to work faster.
The Crack
Three incidents in one month. Proprietary source code. Confidential meeting notes. Semiconductor fabrication data. All in OpenAI's pipeline. No policy. No audit trail. No governance.
The Correction
Samsung banned ChatGPT, began building internal AI. Real lesson: shadow AI runs everywhere. The absence of governance is itself a strategic failure.
