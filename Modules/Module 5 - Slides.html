Lecture · Point 3 of 4

Agent Governance

Software picks actions—not just steps—lock these four.
Autonomy
Draft ≠ send · read ≠ write—explicit space
Tool Calls
Whitelist · rate-limit · log—each call = exfil risk
Memory
What persists · TTL · who reads—HIPAA/regs
Chain
B fails on A—named owner per handoff

Microsoft Copilot + Graph/plugins = production orchestration—Gokul: principles beat model names. Point: one tool call that scares Legal if unlogged.

Lecture · Point 4 of 4

Agentic vs. Traditional ROI

Same word “automation,” different economics + risk.
Traditional
Fixed paths · linear ROI · snaps on edge cases
Agentic
Token $ swings · fuzzy quality · eats exceptions—if governed
Complexity
Simple → scripts · messy → agents
Error $
Low tolerance → review tax kills ROI
Learning
Agents pay off if loops ship

UiPath-style RPA vs. frontier LLM agents—same slide, different P&L + audit trail.

Thumbs: stake a bonus on agent output with zero human gate?
